# Biometric Attendance System

## 📌 Overview
A smart attendance system based on fingerprint recognition using a microcontroller and biometric sensor.  
The system captures, verifies, and records attendance efficiently with minimal human intervention.

## 🚀 Features
- Fingerprint-based authentication
- Fast and reliable attendance marking
- Unique ID mapping for each user
- Real-time data processing
- Low-cost and scalable design

## 🛠️ Technologies Used
- NodeMCU (ESP8266) / Microcontroller
- AS608 Fingerprint Sensor
- Embedded C / Arduino IDE
- Serial Communication

## 🔌 Hardware Components
- NodeMCU / Microcontroller Board
- AS608 Fingerprint Module
- Power Supply
- Connecting Wires

## ⚙️ Working Principle
1. User enrolls fingerprint into the system  
2. Fingerprint data is stored with a unique ID  
3. During attendance, fingerprint is scanned  
4. System matches input with stored data  
5. If matched, attendance is recorded
